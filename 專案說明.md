# 中華電信 115年招考準備系統

## 專案目標

專注準備中華電信 115年基層從業人員遴選「技術類-資訊系統開發及維運」職位。

## 目標職位

**中華電信股份有限公司**
- 技術類 - 資訊系統開發及維運
- 職階：專業職(四)工程師
- 類組：0381（台北/新北）、0354（新竹）、0356（台中）

## 準備重點

### 1. 考試資訊
- 報名：2026/01/02 - 02/02
- 筆試：2026/03/14
- 口試：2026/05/16-17
- 錄取公告：2026/05/26

### 2. 考試科目（筆試佔60%、口試佔40%）

#### 普通科目（佔筆試20%）
- 英文
- 邏輯思維

#### 專業科目（佔筆試80%）★★★★★
- 計算機概論
- 程式設計概論

### 3. 核心學習內容

#### 3.1 計算機概論（最重要★★★★★）
1. **計算機組織與結構**
   - CPU、記憶體階層、匯流排

2. **作業系統**
   - 程序管理、記憶體管理
   - 死結（Deadlock）
   - 檔案系統、排程演算法

3. **資料結構**
   - 樹（Tree）、圖（Graph）
   - 雜湊表（Hash Table）
   - 堆疊（Stack）、佇列（Queue）

4. **資料庫系統**
   - 正規化（1NF~BCNF）
   - SQL 語法
   - 交易管理（ACID）

5. **網路概論**
   - OSI 七層模型
   - TCP/IP 協定

6. **新興科技**（中華電信特別重視）
   - 5G 技術
   - IoT 物聯網
   - 雲端運算

#### 3.2 程式設計概論（最重要★★★★★）
- **物件導向程式設計（OOP）**
  - 封裝、繼承、多型
  - 類別與物件

- **資料結構實作**
  - 鏈結串列、堆疊、佇列
  - 二元搜尋樹

- **演算法**
  - 排序演算法（Quick Sort, Merge Sort）
  - 搜尋演算法（Binary Search）

### 4. 學習資源

#### 推薦書籍
- 《計算機概論》- 旗標出版
- 《作業系統》- Silberschatz（恐龍書）
- 《資料結構》- 蔡明志
- 《演算法圖鑑》

#### 線上練習
- LeetCode - 程式題練習
- HackerRank - 演算法練習
- Zerojudge - 基礎題練習

#### 考古題資源
- 鼎文出版社 - 中華電信歷年試題
- 千華出版社 - 中華電信題庫
- PTT 國考版、中華電信版

### 5. 準備策略

#### 時間分配建議（3-6個月）
1. **第一階段（1-2個月）**
   - 計算機概論基礎
   - 程式語言基礎語法

2. **第二階段（2-3個月）**
   - 資料結構深入學習
   - 演算法練習
   - 資料庫系統

3. **第三階段（考前1個月）**
   - 考古題密集練習
   - 新興科技（5G、IoT、雲端）
   - 口試準備

#### 重點提醒
⚠️ 專業科目總成績未達 50 分，不得參加第二試
⚠️ 口試成績須達 70 分以上才予錄取
⚠️ 須具備正常辨色力（色弱或色盲者不宜報考）
