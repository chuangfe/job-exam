# Q26 型別轉換（Type Coercion）—— 答案

## 輸出

```
"123"
3
15
""
2
```

## 解析

- `1 + "2" + 3`：只要有字串，`+` 就變成字串串接 → `1 + "2"` = `"12"`，再 `+ 3` = `"123"`。
- `"5" - 2`：`-` 沒有字串語意，字串會被轉成數字 → `5 - 2` = `3`。
- `10 + +"5"`：一元正號 `+"5"` 把字串轉成數字 5，所以 `10 + 5` = `15`。
- `[] + []`：陣列轉成字串都是空字串 `""`，相加 = `""`。
- `true + 1`：`true` 轉成數字 1，`1 + 1` = `2`。

口訣：**`+` 遇字串就串接，`-` `*` `/` 一律轉數字。**
