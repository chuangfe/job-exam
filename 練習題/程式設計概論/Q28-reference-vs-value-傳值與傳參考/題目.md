# Q28 傳值與傳參考（Value vs Reference）

請預測以下每個 `console.log` 的輸出結果：

```javascript
let a = 10;
let b = a;
b = 20;
console.log(a);          // ?

let obj1 = { x: 1 };
let obj2 = obj1;
obj2.x = 99;
console.log(obj1.x);     // ?

function change(o) { o.x = 5; }
const p = { x: 1 };
change(p);
console.log(p.x);        // ?
```

問：輸出是什麼？
