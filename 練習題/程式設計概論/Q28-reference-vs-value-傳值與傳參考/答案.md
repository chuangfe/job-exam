# Q28 傳值與傳參考（Value vs Reference）—— 答案

## 輸出

```
10
99
5
```

## 解析

- 基本型別（`number`）是**傳值**：`b = a` 複製了值，之後改變 `b` 不會影響 `a`，所以 `a` 仍是 10。
- 物件是**傳參考**：`obj2 = obj1` 讓兩者指向同一個物件，改 `obj2.x` 等於改 `obj1.x`，所以印出 99。
- 函式參數同理：傳進去的是參考，函式內 `o.x = 5` 修改的是同一個物件的屬性，會影響外部，所以 `p.x` 變成 5。

傳值 vs 傳參考是高頻考點：基本型別複製值，物件（含陣列、函式）複製的是參考。
