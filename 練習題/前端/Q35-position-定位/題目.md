# CSS position（定位）

## 問題
1. CSS `position` 有哪些值？分別是 `static`、`relative`、`absolute`、`fixed`、`sticky`。
2. 這幾個值在「定位基準」與「是否佔據原本空間」上有什麼差異？
3. `absolute` 是相對誰定位？要如何指定它的定位基準（containing block）？
4. `sticky` 的行為是怎麼運作的？

## 引導思考
- `relative` 偏移後，原本的位置還會不會被保留？
- 想用 `absolute` 把子元素定位在某個父層之內，父層需要做什麼設定？
- `fixed` 與 `sticky` 都會「黏住」，它們的差別在哪裡？
