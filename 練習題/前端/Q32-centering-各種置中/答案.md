# CSS 各種置中方式解答

## 概念
置中可分為「水平置中」「垂直置中」與「兩者同時」。現代開發以 **flex / grid** 為首選，最簡潔且不需知道元素尺寸；傳統做法（絕對定位 + transform、line-height）在特定情境仍實用。

## 程式碼示範
```css
/* flex 置中（最常用） */
.parent { display: flex; justify-content: center; align-items: center; }

/* grid 置中（更精簡） */
.parent { display: grid; place-items: center; }

/* 絕對定位 + transform（不需知道尺寸） */
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 行內/單行文字垂直置中 */
.box { height: 40px; line-height: 40px; text-align: center; }
```

## 解析
- **flex 置中**：`justify-content` 控制主軸（預設水平），`align-items` 控制交叉軸（預設垂直），兩者一起設 `center` 即水平+垂直置中。最常用、最直覺。
- **grid 置中**：`place-items: center` 是 `align-items` 與 `justify-items` 的簡寫，一行搞定雙向置中。
- **絕對定位 + transform**：`top/left: 50%` 會讓元素「左上角」落在中心，再用 `translate(-50%, -50%)` 把元素往回拉自身一半寬高，因此**不需事先知道子元素尺寸**即可精準置中。
- **line-height 置中**：把 `line-height` 設成與容器 `height` 相同，可讓**單行文字**垂直置中，搭配 `text-align: center` 水平置中。注意僅適用單行文字。
