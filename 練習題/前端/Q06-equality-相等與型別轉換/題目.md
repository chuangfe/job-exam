# == vs === 與型別轉換

## 問題

請說明以下重點，並用程式碼佐證：

1. `===`（嚴格相等）和 `==`（寬鬆相等）的核心差異是什麼？
2. `==` 在做型別轉換時有哪些常見規則？（null/undefined、數字與字串、布林、物件與原始型別）
3. 為什麼 `NaN === NaN` 是 `false`？該如何正確判斷 `NaN`？
4. 實務上應該優先使用哪一種？什麼情況可以用 `==`？

請思考下列各式的結果：

```js
0 == '';
0 == '0';
'' == '0';
null == 0;
[] == ![];
NaN == NaN;
```
