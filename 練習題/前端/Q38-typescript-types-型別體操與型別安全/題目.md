## Q7.（挑戰）手刻 Utility Types
請用 mapped types + 條件型別 + `infer` 自己實作下列型別，並讓測試用例成立：

```ts
// (a) MyPick：從 T 中挑出 K 這些 key
type R1 = MyPick<{ a: 1; b: 2; c: 3 }, 'a' | 'c'>;
// 期望：{ a: 1; c: 3 }

// (b) MyPartial：把所有屬性變成可選
type R2 = MyPartial<{ a: 1; b: 2 }>;
// 期望：{ a?: 1; b?: 2 }

// (c) MyReadonly：把所有屬性變成唯讀
type R3 = MyReadonly<{ a: 1 }>;
// 期望：{ readonly a: 1 }

// (d) DeepReadonly：巢狀也要全部唯讀
type R4 = DeepReadonly<{ a: { b: { c: 1 } } }>;
// 期望：{ readonly a: { readonly b: { readonly c: 1 } } }

// (e) MyReturnType：取出函式回傳型別
type R5 = MyReturnType<() => string>;
// 期望：string
type R6 = MyReturnType<(x: number) => { ok: boolean }>;
// 期望：{ ok: boolean }
```
