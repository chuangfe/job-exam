# Q26 - 跨頁 / 跨分頁溝通

## 問題

在瀏覽器中，要讓不同分頁（tab）、視窗或 iframe 之間互相溝通有哪些方式？各自適用什麼情境，有什麼限制？

## 引導思考

- 「同源的多個分頁」要即時互相通知，最直接的 API 是什麼？
- `localStorage` 搭配什麼事件可以做到跨分頁監聽？這個事件有什麼特性（觸發它的分頁自己會收到嗎）？
- 跨來源（cross-origin）的 iframe 或彈出視窗要溝通，要用哪個 API？使用時為什麼一定要驗證 `origin`？
- 什麼情況下會選用 `SharedWorker` 或 `Service Worker`？它們能提供哪些上面方法做不到的能力？
- 哪些方法限定「同源」？哪些可以「跨來源」？
