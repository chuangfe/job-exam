# this 綁定

## 問題

請回答以下問題，並用程式碼說明你的理解：

1. `this` 的綁定規則有哪些？它們的優先序由高到低分別是什麼？
2. 預設綁定在嚴格模式與非嚴格模式下有什麼差異？
3. `call`、`apply`、`bind` 三者有什麼差異？分別在什麼情況使用？
4. 箭頭函式的 `this` 有什麼特性？能不能用 `call` / `apply` / `bind` 改變它？

請思考下列程式碼各會回傳什麼：

```js
const obj = { x: 1 };
function f(a, b) { return this.x + a + b; }
f.call(obj, 2, 3);
f.apply(obj, [2, 3]);
const g = f.bind(obj, 2);
g(3);
```
