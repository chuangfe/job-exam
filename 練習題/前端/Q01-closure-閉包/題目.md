# Closure（閉包）

## 問題

請說明以下幾個重點，並嘗試用程式碼佐證你的理解：

1. 什麼是閉包（Closure）？它和函式的「詞法環境（lexical environment）」有什麼關係？
2. 為什麼內層函式在外層函式回傳後，仍然能存取外層作用域的變數？這對記憶體（GC）有什麼影響？
3. 閉包在實務上有哪些常見的用途？
4. 在迴圈中用 `var` 註冊回呼函式時，常見的陷阱是什麼？又該如何解決？

請思考下列程式碼會印出什麼，以及為什麼：

```js
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 0);
}
```
