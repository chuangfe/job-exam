# 修正 useEffect 無限迴圈

## 題目
以下元件一執行就會陷入無限迴圈，畫面上的數字會無止盡地往上加。請找出「為什麼會無限迴圈」，並改寫成正確版本。

```tsx
// 錯誤版
function Buggy() {
  const [data, setData] = useState({ count: 0 });
  useEffect(() => {
    setData({ count: data.count + 1 });
  });
  return <div>{data.count}</div>;
}
```

## 引導思考
- 這個 `useEffect` 有沒有依賴陣列？沒有依賴陣列時，effect 在什麼時候會執行？
- effect 內呼叫了 `setData`，這會造成什麼後果？接著又會發生什麼？
- 把上面兩點串起來，描述出完整的循環。
- 修正方向：
  - 要不要補上依賴陣列？
  - effect 內無條件 setState 是否合理？能否加上「終止條件」？
  - 想想還有哪些常見寫法也會引發無限迴圈（例如依賴陣列裡放了每次渲染都新建的物件/陣列/函式）。
