# 原型與原型鏈（prototype chain）

## 問題

請說明以下重點，並用程式碼佐證：

1. 函式的 `prototype` 屬性與物件的內部 `[[Prototype]]` 有什麼關係？如何存取後者？
2. 存取一個物件的屬性時，JavaScript 如何沿著原型鏈查找？查找的終點是什麼？
3. `new` 運算子實際上做了哪些事？
4. `class` 和原型繼承有什麼關係？`extends` 與 `super` 各做了什麼？
5. `instanceof` 是如何判斷的？

請思考下列程式碼的結果：

```js
function Animal(name) { this.name = name; }
Animal.prototype.speak = function () { return `${this.name} makes a sound`; };
const a = new Animal('cat');
a.speak();
Object.getPrototypeOf(a) === Animal.prototype;
a instanceof Animal;
```
