# 中華電信 - 考試準備策略

## 🎯 準備總覽

### 考試特色分析

#### 優勢
- ✅ 專業科目只有2科（計概 + 程式設計）
- ✅ 題型多為選擇題，較易準備
- ✅ 考試範圍明確
- ✅ 歷年考古題可參考

#### 挑戰
- ⚠️ 計算機概論範圍廣泛
- ⚠️ 需要實作程式碼（非選擇題）
- ⚠️ 新興科技題目（5G、IoT）佔比高
- ⚠️ 競爭激烈，錄取率低

### 投資報酬率分析

## 📅 準備時程規劃

### 全職考生（3-4個月）

#### 第一個月：基礎建立
**週1-2：計算機概論 - 基本概念與數字系統**
- 每天8小時
- 閱讀教科書第1-2章
- 練習進位轉換

**週3-4：資料結構 - 基本結構**
- 每天8小時
- 學習陣列、鏈結串列、堆疊、佇列
- 手寫實作練習

**週末：程式設計基礎**
- 複習程式語法
- 練習基本題目

#### 第二個月：重點科目
**週1-2：作業系統**
- 程序管理
- 記憶體管理
- 檔案系統
- 死結處理

**週3：資料庫系統**
- ER Model
- 正規化
- SQL語法
- 交易管理

**週4：網路概論**
- OSI模型
- TCP/IP
- 路由與交換

**週末：程式設計進階**
- 物件導向概念
- 常見演算法實作

#### 第三個月：深入學習
**週1：計算機組織**
- CPU結構
- 記憶體階層
- I/O系統

**週2：資訊安全**
- 密碼學基礎
- 常見攻擊
- 防護機制

**週3：新興科技**
- 5G技術
- 雲端運算
- IoT、AI基礎

**週4：演算法與資料結構進階**
- 樹狀結構
- 圖論
- 排序演算法

**週末：考古題練習**
- 歷年試題
- 模擬考

#### 第四個月：衝刺階段
**週1-2：大量練習考古題**
- 每天完整模擬考
- 檢討錯題
- 整理重點筆記

**週3：弱項加強**
- 針對錯誤率高的章節
- 重新複習教科書
- 做專題練習

**週4：考前總複習**
- 複習所有重點筆記
- 背誦重要公式
- 調整作息

**考前一週：輕鬆複習**
- 看筆記為主
- 不碰新內容
- 保持最佳狀態

## 📚 各科準備策略

### 1. 計算機概論（45%時間）

#### 準備優先序
```
第一優先（必讀）：
- 資料結構 ★★★★★
- 作業系統 ★★★★★
- 資料庫系統 ★★★★★
- 網路概論 ★★★★★

第二優先（重要）：
- 計算機組織 ★★★★☆
- 資訊安全 ★★★★☆
- 新興科技（5G/IoT/雲端） ★★★★☆

第三優先（次要）：
- 數字系統 ★★★☆☆
- 布林代數 ★★★☆☆
```

#### 讀書方法

**第一輪：建立架構（40%時間）**
1. 通讀教科書
2. 理解基本概念
3. 製作心智圖
4. 標記重點

**第二輪：深入理解（40%時間）**
1. 精讀重點章節
2. 練習課後習題
3. 觀看教學影片
4. 整理公式與定理

**第三輪：實戰演練（20%時間）**
1. 練習考古題
2. 檢討錯誤
3. 補強弱點

#### 各章節準備技巧

**資料結構**
- 手繪圖形理解
- 實作程式碼加深印象
- 比較不同資料結構優缺點
- 記憶時間複雜度

**作業系統**
- 理解每個演算法的運作方式
- 能手算排程結果
- 畫圖理解死結
- 記憶重要公式（Effective Access Time等）

**資料庫系統**
- 實際操作SQL語法
- 練習正規化過程
- 畫ER Diagram
- 理解交易ACID特性

**網路概論**
- 記憶OSI七層與TCP/IP四層
- 了解每層的協定
- 計算子網路遮罩
- 理解路由原理

**計算機組織**
- 理解CPU運作流程
- 記憶記憶體階層
- 計算Cache命中率
- 了解管線化

**資訊安全**
- 區分對稱與非對稱加密
- 了解常見攻擊手法
- 記憶防護機制
- 關注時事資安事件

**新興科技**
- 關注5G三大應用場景（eMBB, mMTC, uRLLC）
- 理解雲端服務模型（IaaS, PaaS, SaaS）
- 了解IoT架構
- 認識AI/ML基本概念

#### 記憶技巧

**使用口訣**
- OSI七層：「請不要丟掉我的披薩」（實、資、網、傳、會、表、應）
- ACID：「原一隔持」（Atomicity, Consistency, Isolation, Durability）

**製作表格比較**
| 項目 | TCP | UDP |
|------|-----|-----|
| 可靠性 | 可靠 | 不可靠 |
| 連接 | 需建立連接 | 無連接 |
| 速度 | 慢 | 快 |

**圖像記憶**
- 畫流程圖理解演算法
- 用圖表比較不同方法

### 2. 程式設計概論（35%時間）

#### 準備優先序
```
第一優先：
- 基本語法熟練 ★★★★★
- 資料結構實作 ★★★★★
- 排序演算法 ★★★★★

第二優先：
- 物件導向程式設計 ★★★★☆
- 常見演算法 ★★★★☆
- 遞迴問題 ★★★★☆

第三優先：
- 進階資料結構 ★★★☆☆
```

#### 語言選擇建議

**推薦Java**
- ✅ 物件導向特性完整
- ✅ 語法嚴謹，適合考試
- ✅ 教材豐富
- ❌ 語法較冗長

**推薦C++**
- ✅ 執行效率高
- ✅ 適合演算法實作
- ✅ STL功能強大
- ❌ 語法較複雜

**推薦Python**
- ✅ 語法簡潔
- ✅ 內建資料結構豐富
- ✅ 易於快速開發
- ❌ 手寫考試可能較吃虧（若題目要求效率）

**建議**：選擇你最熟悉的語言即可

#### 練習方法

**每日練習（至少3題）**
```
早上：1題簡單題（20分鐘）
中午：1題中等題（30分鐘）
晚上：1題困難題（40分鐘）
```

**週末專題（2-3小時）**
- 實作完整資料結構（BST, Heap等）
- 大型演算法問題
- LeetCode Medium/Hard題目

#### 必練題型

**基礎題（前1個月）**
1. 陣列操作（反轉、旋轉、搜尋）
2. 字串處理（回文、字串比對）
3. 數學運算（質數、GCD、LCM）
4. 簡單遞迴（階乘、費式數列）

**進階題（第2-3個月）**
1. 堆疊應用（括號配對、表示式轉換）
2. 佇列應用（約瑟夫問題）
3. 鏈結串列操作（反轉、合併、檢測環）
4. 二元樹遍歷（前中後序、層序）
5. 排序演算法實作
6. 搜尋演算法實作
7. 動態規劃基礎

**挑戰題（考前衝刺）**
1. 圖論問題（DFS, BFS, 最短路徑）
2. 複雜資料結構（AVL, Red-Black Tree）
3. 進階演算法（回溯、分治）

#### 練習平台

**推薦順序**
1. **Zerojudge**（台灣中學生常用，有中文題目）
2. **APCS歷屆試題**（很接近考試難度）
3. **LeetCode Easy**（熟悉後進階到Medium）
4. **HackerRank**（有各種主題練習）

#### 手寫程式技巧

**考試特別注意**
- ✍️ 字跡工整
- ✍️ 使用偽碼或註解說明思路
- ✍️ 先處理主要邏輯，再補充邊界條件
- ✍️ 變數命名有意義
- ✍️ 適當縮排

**常見失分點**
- ❌ 陣列越界
- ❌ 指標使用錯誤
- ❌ 邊界條件未處理
- ❌ 邏輯錯誤
- ❌ 語法錯誤

**減少錯誤的方法**
1. 寫完後自己測試幾個案例
2. 特別檢查邊界條件（空陣列、單一元素等）
3. 確認變數初始化
4. 檢查迴圈終止條件

## 💡 準備技巧與心態

### 讀書技巧

#### 1. 番茄鐘工作法
```
25分鐘專注 → 5分鐘休息 → 重複4次 → 長休息15-30分鐘
```

#### 2. 費曼學習法
```
1. 選擇概念
2. 用自己的話解釋
3. 找出不懂的地方
4. 重新學習與簡化
```

#### 3. 間隔複習法
```
第1天學習 → 第2天複習 → 第4天複習 → 第7天複習 → 第15天複習
```

### 時間管理

#### 優先序矩陣
```
重要且緊急 → 立刻做（考古題練習、弱點加強）
重要不緊急 → 計劃做（新章節學習、基礎建立）
緊急不重要 → 委託做（可省略的雜事）
不重要不緊急 → 不要做（娛樂、社交暫緩）
```

### 壓力管理

#### 適度運動
- 每天30分鐘運動
- 提升專注力
- 舒緩壓力

#### 充足睡眠
- 每天至少7小時
- 記憶力更好
- 學習效率高

#### 正向心態
- 相信自己可以做到
- 不與他人比較
- 專注自己的進步

## ⚠️ 常見錯誤

### 準備階段

❌ **錯誤1：只讀不練**
- 計算機概論需要理解，不能死背
- 程式設計一定要動手寫

✅ 正確做法：讀完立刻練習題目

❌ **錯誤2：完美主義**
- 想把一個章節讀到100分才進下一章
- 結果時間不夠

✅ 正確做法：先快速掃過一遍，再重點精讀

❌ **錯誤3：忽略考古題**
- 只讀教科書，不練習考古題
- 不熟悉考試題型

✅ 正確做法：考前2個月大量練習考古題

❌ **錯誤4：偏科準備**
- 只讀自己擅長的科目
- 弱科放棄

✅ 正確做法：弱科多花時間，強科保持水準

### 考試當天

❌ **錯誤1：時間分配不當**
- 卡在難題太久
- 簡單題沒時間寫

✅ 正確做法：先寫會的，難題標記後回頭寫

❌ **錯誤2：緊張過度**
- 看到不會的題目就慌張
- 影響後續作答

✅ 正確做法：深呼吸，冷靜思考

## 🎯 考試當天

### 前一晚
- 早點睡（10-11點）
- 不要再讀書
- 準備好明天的物品

### 當天早上
- 提早30分鐘到考場
- 吃適量早餐
- 上廁所
- 深呼吸放鬆

### 考試中
1. **拿到試卷先瀏覽**
   - 看題目分布
   - 估算時間分配

2. **作答策略**
   - 選擇題先寫會的
   - 不會的先跳過
   - 計算題寫出解題過程
   - 程式題先寫註解說明思路

3. **時間控制**
   - 每30分鐘檢查一次時間
   - 預留15分鐘檢查

4. **檢查答案**
   - 確認有塗滿
   - 檢查計算錯誤
   - 確認沒有漏題
