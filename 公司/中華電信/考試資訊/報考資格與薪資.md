## 💰 薪資待遇

### 起薪

#### 2026年標準
- **起薪**：50,020元/月
- **職等**：工務類專業職（四）第一類專員

#### 歷年調薪記錄
| 年度 | 起薪（元/月） | 調幅 |
|------|--------------|------|
| 2026 | 50,020 | - |
| 2024 | 47,000左右 | - |
| 2022 | 45,000左右 | - |

*註：中華電信定期調整薪資，以官方公告為準*

### 薪資組成

#### 每月固定薪資
```
月薪 = 本薪 + 職務加給 + 伙食津貼 + 交通津貼
```

#### 年度獎金
1. **年終獎金**
   - 約3.5 - 4.4個月（依公司營運狀況）
   - 依年度績效評估發放

2. **績效獎金**
   - 依個人考績
   - 依部門績效
   - 通常每年中發放

3. **三節獎金**
   - 端午節、中秋節、春節
   - 合計約2-3個月薪資

### 年薪試算

#### 第一年（保守估計）
```
年薪 = 月薪 × 12 + 年終獎金 + 績效獎金 + 三節獎金
     = 50,020 × 12 + (50,020 × 3.5) + (50,020 × 2)
     = 600,240 + 175,070 + 100,040
     = 875,350元
     ≈ 87-90萬
```

#### 第三年（預估）
```
假設每年調薪3-5%
月薪約：52,000 - 55,000元
年薪約：85-95萬元
```

#### 第五年（預估）
```
假設持續調薪並晉升
月薪約：55,000 - 60,000元
年薪約：95-105萬元
```

### 調薪機制

#### 年度調薪
- **頻率**：每年7月
- **依據**：個人考績、部門績效、公司營運狀況
- **幅度**：通常3-5%（視考績等級）

#### 晉升調薪
- 晉升至高級專員：薪資約增加10-15%
- 晉升至主管職：薪資約增加20-30%

#### 考績等級
- 甲等：調薪幅度較大
- 乙等：標準調薪
- 丙等：調薪幅度較小或不調薪

#### 退休金
- 採勞退新制
- 公司提撥6%

#### 退休年齡
- 法定退休年齡：65歲
- 可申請提早退休（依公司規定）

## ❓ 常見問題

### Q1：應屆畢業生可以報考嗎？
A：可以。但錄取後需於報到時繳交畢業證書。

### Q2：研究所學歷有加分嗎？
A：簡章未明文規定加分，但面試時可能是加分項。起薪仍依職等標準。

### Q3：非相關科系可以報考嗎？
A：不行。需為理工、資訊、資管相關科系。

### Q4：有工作經驗會加分嗎？
A：筆試成績不會因工作經驗加分，但面試時是優勢。

### Q5：錄取後一定要去嗎？
A：錄取後若放棄，依簡章規定辦理。建議確定要任職再報考。

### Q6：試用期多久？
A：通常為3-6個月，試用期間薪資相同。

### Q7：可以兼職嗎？
A：依公司規定，通常需申請核准。

### Q8：有分發限制嗎？
A：錄取時會告知工作地點，通常需配合公司分發。

### Q9：可以申請調動嗎？
A：到職一定期間後，可依公司制度申請內部調動。

### Q10：退休金制度如何？
A：採勞退新制，公司提撥6%，另有舊制年資者保留舊制權益。
