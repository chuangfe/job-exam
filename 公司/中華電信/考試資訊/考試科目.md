# 中華電信 - 考試科目詳細說明

## 📚 考試科目總覽

### 資訊系統開發及維運職位

中華電信技術類考試分為兩大部分：

#### 第一試：筆試（佔總成績60%）
1. **共同科目（佔筆試成績20%）**
   - 英文
   - 邏輯思維

2. **專業科目（佔筆試成績80%）**
   - 計算機概論
   - 程式設計概論

#### 第二試：口試（佔總成績40%）
- 個人面試
- 時間約10-15分鐘
- **重要**：口試成績未達70分不予錄取

   - 時態
   - 動詞變化
   - 介系詞用法

   - 假設語氣

3. **閱讀測驗**
   - 短文理解
   - 段落填空
   - 主旨判斷
   - 細節理解

4. **技術相關英文**
   - 資訊科技術語
   - 電信相關詞彙
   - 商業英文

#### 準備方向
- 📚 背誦常用英文單字（尤其是資訊相關）
- 📚 練習多益或全民英檢中高級題目
- 📚 閱讀英文科技文章（TechCrunch、The Verge等）
- 📚 熟悉5G、IoT、Cloud、AI等技術英文

#### 推薦資源
- 多益官方題庫
- 全民英檢中高級題庫
- 科技新聞網站
- 資訊專業英文字典

## 💻 專業科目

### 1. 計算機概論（Computer Fundamentals）

#### 考試方式
- **題型**：選擇題 + 非選擇題（問答題、計算題）
- **比重**：專業科目50%
- **難度**：中上
- **範圍**：非常廣泛

#### 主要章節

##### 📌 第一章：基本概念
- 計算機發展史
- 電腦分類（超級電腦、大型電腦、個人電腦等）
- 數字系統（二進位、八進位、十六進位）
- 數字轉換與運算
- 布林代數

##### 📌 第二章：計算機組織與結構
**重要度：★★★★★**

1. **中央處理單元（CPU）**
   - CPU組成：CU（控制單元）、ALU（算術邏輯單元）
   - 暫存器（Register）
   - 快取記憶體（Cache）
   - 管線化（Pipeline）

2. **記憶體階層**
   - 暫存器
   - Cache（L1, L2, L3）
   - 主記憶體（RAM, ROM）
   - 輔助記憶體（HDD, SSD）

3. **匯流排（Bus）**
   - 資料匯流排
   - 位址匯流排
   - 控制匯流排

4. **輸入輸出裝置**
   - I/O介面
   - DMA（直接記憶體存取）
   - 中斷機制

##### 📌 第三章：作業系統
**重要度：★★★★★**

1. **作業系統基本概念**
   - 作業系統功能
   - 作業系統類型（批次、分時、即時）
   - 核心（Kernel）vs 使用者模式

2. **程序管理（Process Management）**
   - 程序（Process）vs 執行緒（Thread）
   - 程序狀態（New, Ready, Running, Waiting, Terminated）
   - 程序排程演算法：
     - FCFS（First-Come, First-Served）
     - SJF（Shortest Job First）
     - Priority Scheduling
     - Round Robin
     - Multilevel Queue

3. **記憶體管理**
   - 邏輯位址 vs 實體位址
   - 分頁（Paging）
   - 分段（Segmentation）
   - 虛擬記憶體（Virtual Memory）
   - 分頁置換演算法：
     - FIFO
     - LRU（Least Recently Used）
     - Optimal

4. **死結（Deadlock）**
   - 死結的四個必要條件
   - 死結預防
   - 死結避免（Banker's Algorithm）
   - 死結偵測與恢復

5. **檔案系統**
   - 檔案組織方式
   - 目錄結構
   - 檔案存取方法
   - 磁碟排程（FCFS, SSTF, SCAN, C-SCAN）

##### 📌 第四章：資料結構
**重要度：★★★★★**

1. **基本資料結構**
   - 陣列（Array）
   - 鏈結串列（Linked List）：單向、雙向、環狀
   - 堆疊（Stack）：LIFO特性、應用
   - 佇列（Queue）：FIFO特性、環狀佇列、優先佇列

2. **樹狀結構（Tree）**
   - 二元樹（Binary Tree）
   - 二元搜尋樹（BST）
   - AVL樹（平衡樹）
   - B樹、B+樹
   - 堆積（Heap）：Max Heap, Min Heap
   - 樹的遍歷：前序、中序、後序、層序

3. **圖（Graph）**
   - 圖的表示法：鄰接矩陣、鄰接串列
   - 圖的遍歷：DFS（深度優先）、BFS（廣度優先）
   - 最短路徑：Dijkstra、Bellman-Ford、Floyd-Warshall
   - 最小生成樹：Prim、Kruskal

4. **雜湊表（Hash Table）**
   - 雜湊函數
   - 碰撞處理：鏈結法、開放定址法

##### 📌 第五章：演算法
**重要度：★★★★☆**

1. **排序演算法**
   - 氣泡排序（Bubble Sort）
   - 選擇排序（Selection Sort）
   - 插入排序（Insertion Sort）
   - 合併排序（Merge Sort）
   - 快速排序（Quick Sort）
   - 堆積排序（Heap Sort）
   - 時間複雜度比較

2. **搜尋演算法**
   - 線性搜尋（Linear Search）
   - 二分搜尋（Binary Search）

3. **複雜度分析**
   - Big O notation
   - 時間複雜度
   - 空間複雜度

##### 📌 第六章：資料庫系統
**重要度：★★★★★**

1. **資料庫基本概念**
   - 資料庫管理系統（DBMS）
   - 三層式架構
   - 資料獨立性

2. **資料模型**
   - ER Model（實體關係模型）
   - 關聯式資料模型

3. **正規化（Normalization）**
   - 第一正規化（1NF）
   - 第二正規化（2NF）
   - 第三正規化（3NF）
   - BCNF

4. **SQL語言**
   - DDL（CREATE, ALTER, DROP）
   - DML（SELECT, INSERT, UPDATE, DELETE）
   - DCL（GRANT, REVOKE）
   - JOIN操作（INNER, LEFT, RIGHT, FULL）
   - 子查詢（Subquery）
   - 聚合函數（COUNT, SUM, AVG, MAX, MIN）

5. **交易管理**
   - ACID特性
   - Concurrency Control
   - 鎖定機制

6. **索引**
   - B+樹索引
   - 雜湊索引
   - 索引優缺點

##### 📌 第七章：網路概論
**重要度：★★★★★**

1. **網路基礎**
   - OSI七層模型
   - TCP/IP四層模型

2. **實體層與資料鏈結層**
   - 傳輸媒介
   - 編碼方式
   - MAC位址
   - Ethernet
   - 交換器（Switch）

3. **網路層**
   - IP位址（IPv4, IPv6）
   - 子網路遮罩
   - 路由協定（RIP, OSPF, BGP）
   - ICMP

4. **傳輸層**
   - TCP vs UDP
   - 三方交握（Three-way Handshake）
   - 流量控制
   - 壅塞控制

5. **應用層**
   - HTTP/HTTPS
   - DNS
   - FTP
   - SMTP/POP3/IMAP
   - DHCP

##### 📌 第八章：資訊安全
**重要度：★★★★☆**

1. **資訊安全基本概念**
   - CIA三要素：機密性、完整性、可用性
   - 身份驗證（Authentication）
   - 授權（Authorization）

2. **密碼學**
   - 對稱式加密（AES, DES）
   - 非對稱式加密（RSA）
   - 雜湊函數（MD5, SHA）
   - 數位簽章

3. **常見攻擊**
   - DDoS攻擊
   - SQL Injection
   - XSS（跨站腳本攻擊）
   - CSRF
   - 中間人攻擊

4. **防護機制**
   - 防火牆
   - IDS/IPS
   - VPN
   - SSL/TLS

##### 📌 第九章：新興科技
**重要度：★★★★☆**（中華電信特別重視）

1. **5G技術**
   - eMBB（增強型行動寬頻）
   - mMTC（大規模物聯網通訊）
   - uRLLC（超可靠低延遲通訊）
   - 5G vs 4G差異

2. **雲端運算**
   - IaaS, PaaS, SaaS
   - 公有雲、私有雲、混合雲
   - 虛擬化技術
   - 容器技術（Docker, Kubernetes）

3. **物聯網（IoT）**
   - IoT架構
   - 感測器與致動器
   - IoT通訊協定（MQTT, CoAP）

4. **人工智慧與大數據**
   - 機器學習基本概念
   - 深度學習簡介
   - 大數據3V特性
   - 資料探勘

5. **區塊鏈**
   - 區塊鏈基本概念
   - 去中心化
   - 共識機制

#### 準備策略
1. **基礎先行**：先掌握資料結構、作業系統、資料庫
2. **重點突破**：計算機組織、網路概論、資料結構為重點
3. **新技術關注**：5G、IoT、雲端為中華電信特色，務必準備
4. **考古題練習**：大量練習歷屆試題

### 2. 程式設計概論

#### 考試方式
- **題型**：選擇題 + 非選擇題（程式碼撰寫、程式閱讀、除錯）
- **比重**：專業科目50%
- **語言**：通常不限定語言，但建議使用Java、C++或Python

#### 主要內容

##### 📌 基本程式概念
1. **變數與資料型態**
   - 整數、浮點數、字元、字串
   - 型態轉換
   - 常數與變數

2. **運算子**
   - 算術運算子
   - 比較運算子
   - 邏輯運算子
   - 位元運算子

3. **控制結構**
   - 條件判斷（if, if-else, switch）
   - 迴圈（for, while, do-while）
   - 巢狀結構

##### 📌 函數與模組化
1. **函數定義與呼叫**
2. **參數傳遞**（傳值、傳址）
3. **遞迴（Recursion）**
4. **函數重載（Overloading）**

##### 📌 陣列與字串
1. **一維陣列**
2. **多維陣列**
3. **字串處理**
4. **常見陣列演算法**

##### 📌 物件導向程式設計（OOP）
**重要度：★★★★★**

1. **類別與物件**
   - 類別定義
   - 建構子
   - 解構子
   - 成員變數與成員函數

2. **封裝（Encapsulation）**
   - public, private, protected
   - Getter/Setter

3. **繼承（Inheritance）**
   - 單一繼承
   - 多重繼承
   - 方法覆寫（Override）

4. **多型（Polymorphism）**
   - 編譯時期多型
   - 執行時期多型
   - 虛擬函數

5. **抽象化**
   - 抽象類別
   - 介面（Interface）

##### 📌 資料結構實作
1. **鏈結串列實作**
2. **堆疊實作**
3. **佇列實作**
4. **二元搜尋樹實作**

##### 📌 排序與搜尋演算法實作
1. **實作氣泡排序**
2. **實作快速排序**
3. **實作二分搜尋**

##### 📌 程式閱讀與除錯
1. **找出程式錯誤**
2. **預測程式輸出**
3. **分析程式複雜度**

##### 📌 常見程式題型
1. **陣列操作**
   - 陣列反轉
   - 找最大/最小值
   - 陣列排序

2. **字串處理**
   - 字串反轉
   - 回文判斷
   - 字串比對

3. **數學運算**
   - 質數判斷
   - 最大公因數/最小公倍數
   - 費式數列

4. **遞迴問題**
   - 河內塔
   - 階乘計算
   - 組合排列

5. **資料結構應用**
   - 堆疊應用（括號配對）
   - 佇列應用（約瑟夫問題）
   - 樹的遍歷

#### 準備建議

1. **語言選擇**
   - 建議選擇你最熟悉的語言
   - Java：語法較嚴謹，適合大型專案
   - C++：效能優秀，適合演算法實作
   - Python：語法簡潔，適合快速開發

2. **練習平台**
   - LeetCode（推薦）
   - HackerRank
   - Zerojudge
   - APCS考題

3. **準備重點**
   - 熟練基本語法
   - 能快速實作常見資料結構
   - 大量練習程式題
   - 訓練手寫程式能力

4. **考試技巧**
   - 先寫簡單註解說明演算法
   - 注意邊界條件處理
   - 檢查語法錯誤
   - 測試範例輸入輸出

## 📊 成績計算方式

```
總成績 = 筆試成績 × 60% + 口試成績 × 40%

其中：
筆試成績 = 共同科目 × 20% + 專業科目 × 80%
專業科目 = 計算機概論 × 50% + 程式設計概論 × 50%
```

### 錄取標準
1. 筆試成績需達一定標準（通常60%以上）
2. 口試成績需達70分以上（必要條件）
3. 總成績由高至低錄取
