# 中華電信 - 資訊系統開發及維運職位總覽

## 🏢 公司簡介

### 基本資訊
- **公司名稱**：中華電信股份有限公司 (Chunghwa Telecom Co., Ltd.)
- **成立時間**：1996年7月1日（由交通部電信總局改制）
- **公司性質**：國營企業（政府持股約35%）
- **員工人數**：約34,000人
- **產業類別**：電信服務業

### 業務範圍
- 固網通信服務
- 行動通信服務（4G、5G）
- 寬頻網路服務
- 數據通信服務
- 雲端服務與IDC機房
- IoT物聯網應用
- 資通訊整合解決方案

### 企業願景
- 成為「數位經濟的賦能者」
- 推動5G、AI、大數據等新興科技應用
- 發展智慧城市、智慧醫療、智慧交通等解決方案

## 💼 職位資訊

### 職位名稱
**工務類專業職（四）第一類專員 - 資訊系統開發及維運**

### 工作內容
1. **系統開發**
   - 資訊系統規劃與設計
   - 應用程式開發與維護
   - 系統測試與品質控管
   - 程式碼撰寫與除錯

2. **系統維運**
   - 資訊系統日常營運維護
   - 系統監控與效能調校
   - 故障排除與問題處理
   - 系統備份與災難復原

3. **專案管理**
   - 參與資訊專案規劃與執行
   - 需求分析與文件撰寫
   - 跨部門溝通協調
   - 專案進度追蹤

4. **技術研究**
   - 評估新技術可行性
   - 研究產業技術趨勢
   - 參與技術教育訓練
   - 技術文件撰寫

### 使用技術（參考）
- **程式語言**：Java, Python, C++, JavaScript
- **資料庫**：Oracle, MySQL, PostgreSQL, MongoDB
- **框架**：Spring, Django, React, Vue.js
- **雲端平台**：AWS, Azure, Google Cloud
- **開發工具**：Git, Docker, Kubernetes, Jenkins
- **作業系統**：Linux, Windows Server

### 工作地點
- 主要：台北市、新北市
- 其他：依業務需求可能分發至全台各地

## 💰 薪資福利

### 薪資待遇
- **起薪**：50,020元/月（2026年標準）
- **年終獎金**：約4.4個月（依公司營運狀況）
- **績效獎金**：依個人與部門績效發放
- **調薪機會**：每年依考績調薪

### 預估年薪
- **第1年**：約70-75萬（含年終）
- **第3年**：約80-90萬（含調薪與獎金）
- **第5年**：約90-100萬以上

### 福利制度
1. **法定福利**
   - 勞健保、勞退提撥
   - 特休假、病假、產假等

2. **公司福利**
   - 員工持股信託
   - 優惠通信費率（手機、寬頻）
   - 結婚、生育、喪葬補助
   - 子女教育獎學金
   - 員工旅遊補助

3. **保險福利**
   - 團體保險
   - 意外險
   - 醫療險

4. **其他福利**
   - 員工餐廳或午餐補助
   - 員工宿舍（部分地區）
   - 健康檢查
   - 各類社團活動
   - 教育訓練與進修補助

### 工作環境
- **工時**：週休二日，標準工時制
- **加班**：依專案需求，有加班費
- **休假**：依勞基法規定，另有特殊休假
- **工作氛圍**：國營企業文化，穩定但需適應組織層級

## 📈 職涯發展

### 晉升管道
1. **技術路線**
   - 專員 → 高級專員 → 資深專員 → 技術主管

2. **管理路線**
   - 專員 → 副主任 → 主任 → 經理 → 協理 → 副總經理

### 專業發展
- 可參與大型專案（5G、IoT、雲端）
- 接觸多元技術領域
- 公司提供教育訓練資源
- 可考取相關專業證照（公司補助）

### 轉調機會
- 內部職缺公開招募制度
- 可申請跨部門輪調
- 有機會參與海外專案

## 🎯 適合對象

### 學歷背景
- 理工、資訊學院
- 資訊管理相關科系
- 已取得學士（含）以上學位

### 特質與能力
✅ 喜歡程式設計與系統開發
✅ 邏輯思維清晰，善於解決問題
✅ 對新技術有學習熱忱
✅ 具備良好溝通能力
✅ 能承受專案壓力
✅ 重視工作穩定性
✅ 適應團隊合作環境

### 不太適合
❌ 追求高度自由與彈性工作模式
❌ 無法適應組織層級與流程
❌ 期望短期內快速升遷
❌ 偏好新創公司文化

## 📌 重要提醒

1. **技術更新快速**：電信產業技術變革快，需持續學習
2. **專案性質工作**：常需配合專案時程，可能有加班需求
3. **輪班可能性**：部分維運職位可能需要輪班或待命
4. **組織文化**：國營企業文化，決策流程較長
5. **考試競爭**：錄取率相對較低，需充分準備
