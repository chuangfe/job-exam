## 📝 考古題重點題型

### 計算機概論

#### 作業系統（高頻考點）
```
【範例題】
Q：某系統採用Round Robin排程，時間量子為4ms，
   有三個程序P1(12ms)、P2(8ms)、P3(4ms)依序到達，
   請問P2的完成時間為何？

A：P1(0-4) → P2(4-8) → P3(8-12) → P1(12-16) → P2(16-20) → P1(20-24)
   P2完成時間為20ms
```

#### 資料結構（高頻考點）
```
【範例題】
Q：將中序表示式 A+B*C-D 轉換為後序表示式？

A：ABC*+D-

解析：
- 中序（Infix）：A + B * C - D
- 後序（Postfix）：A B C * + D -
```

#### 資料庫（高頻考點）
```
【範例題】
Q：以下哪個不是正規化的目標？
(A) 減少資料重複
(B) 避免異常現象
(C) 提高查詢效率
(D) 保持資料一致性

A：(C)
解析：正規化主要目標是減少重複、避免異常、保持一致性。
      提高查詢效率有時反而需要反正規化。
```

#### 網路概論（高頻考點）
```
【範例題】
Q：TCP三方交握（Three-way Handshake）的順序為何？
(A) SYN → SYN-ACK → ACK
(B) SYN → ACK → SYN-ACK
(C) ACK → SYN → SYN-ACK
(D) SYN-ACK → SYN → ACK

A：(A)
解析：
1. Client → Server: SYN
2. Server → Client: SYN-ACK
3. Client → Server: ACK
```

#### 新興科技（中華電信特色）
```
【範例題】
Q：5G的三大應用場景不包括下列何者？
(A) eMBB（增強型行動寬頻）
(B) mMTC（大規模物聯網通訊）
(C) uRLLC（超可靠低延遲通訊）
(D) vRAN（虛擬無線接取網路）

A：(D)
解析：5G三大應用場景為eMBB、mMTC、uRLLC。
      vRAN是5G的技術架構，不是應用場景。
```

### 程式設計

#### 基礎語法題
```java
【範例題】
Q：以下程式輸出為何？

int[] arr = {1, 2, 3, 4, 5};
int sum = 0;
for(int i = 0; i < arr.length; i += 2) {
    sum += arr[i];
}
System.out.println(sum);

A：9
解析：i=0時arr[0]=1, i=2時arr[2]=3, i=4時arr[4]=5
      1+3+5=9
```

#### 演算法實作題
```python
【範例題】
Q：實作一個函數判斷整數是否為質數

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True
```

#### OOP觀念題
```
【範例題】
Q：請說明封裝（Encapsulation）、繼承（Inheritance）、
   多型（Polymorphism）的意義與差異。

A：
1. 封裝：將資料與方法包裝在類別中，隱藏內部細節，
         透過public/private控制存取權限。

2. 繼承：子類別繼承父類別的屬性與方法，
         達到程式碼重用與擴充。

3. 多型：相同介面可有不同實作，
         如方法覆寫（Override）與方法重載（Overload）。
```
