# 台積電 - IT / 資訊系統工程師 總覽

> 範本類型：**民間企業**（履歷 + 技術面試制）。內容以**台積電官方職缺說明 (JD) 與招募官網為準**，本檔僅供準備參考。

## 🏢 公司簡介
- **公司名稱**：台灣積體電路製造股份有限公司（Taiwan Semiconductor Manufacturing Company, TSMC）
- **產業 / 規模**：半導體晶圓代工（Foundry）龍頭，全球市占第一；員工數逾 7 萬人，屬大型上市公司（台股權值王、ADR 於美股掛牌）。
- **企業文化**：核心價值為**誠信正直 (Integrity)、承諾 (Commitment)、創新 (Innovation)、客戶信任 (Customer Trust)**；強調紀律、流程嚴謹、高度重視工程實務與資安。IT/資訊組織龐大，需支撐先進製程的智慧製造與全球營運系統。

## 💼 目標職缺（JD）
> 以下為 IT/資訊類常見職缺方向，實際職稱與條件請以官方 JD 為準。

- **職位名稱**：IT Engineer / 資訊系統工程師（含軟體工程師、資料工程師、SAP/ERP 顧問、資安工程師、智慧製造/自動化工程師等子類）
- **工作內容**（依子類不同，常見）：
  - 開發與維運企業資訊系統（製造、供應鏈、財務、HR 等）。
  - 設計、開發與維護內部應用程式（Web/後端服務、自動化工具）。
  - 建置與維護資料庫、資料管線（Data Pipeline）、報表與數據分析平台。
  - 整合製造現場系統（MES）、ERP（SAP）等核心系統。
  - 系統效能調校、問題排除、資安防護與合規。
  - 智慧製造：以軟體/演算法/AI 提升產線自動化與良率。
- **必備條件 (Requirements)**（典型）：
  - 資工/資管/電機/數學/統計等相關科系學士以上。
  - 至少一種程式語言扎實基礎（Python / Java / C# / C++ 擇一）。
  - SQL 與關聯式資料庫概念。
  - 良好的問題分析、溝通與團隊合作能力。
  - 願意配合廠區（新竹/台南/台中）工作。
- **加分條件 (Nice to have)**：
  - 雲端（AWS/Azure/GCP）、容器（Docker/K8s）、CI/CD 經驗。
  - 資料工程（ETL、Spark、Kafka）、資料科學/機器學習。
  - SAP/ERP、MES 製造系統、半導體製程相關背景。
  - 資安證照（如 CISSP）或網路/系統管理經驗。
  - 英文讀寫能力（跨國協作、技術文件）。

## 🧰 技術棧（依 JD 整理）
- **語言**：Python、Java、C#（.NET）、C/C++、JavaScript（前端視職缺）；Shell Script。
- **框架/工具**：Spring（Java）、.NET、前端框架（React/Angular 視職缺）；Git、Jenkins/GitLab CI、Docker/Kubernetes、Linux。
- **資料庫/雲端**：Oracle、Microsoft SQL Server、PostgreSQL；資料工程（Spark/Kafka/ETL）；雲端 AWS/Azure/GCP。
- **製造業特有**：SAP（ERP）、MES（製造執行系統）、半導體相關 Domain Knowledge。

## 💰 薪資福利（參考）
> 精確薪資**依學經歷面議**，請以 offer 為準，以下為公開印象，**待查證**。

- **起薪/薪資**：依學經歷面議；台積電薪資在台灣業界**偏高**（不在此捏造精確數字）。
- **分紅/獎金**：員工**分紅（現金獎金）** 為薪酬重要組成，與公司獲利連動。
- **員工認股**：提供**員工認股**相關制度。
- **其他福利**：團體保險、健檢、員工餐廳/交通、進修補助等（依公司規定）。

## 🔄 招募流程
> 流程依職缺/部門略有差異，以官方通知為準。常見如下：

1. **履歷投遞**（台積電招募官網 / 校園徵才 / 內部推薦）。
2. **線上測驗 / 性向測驗**（邏輯、數理、英文等；部分職缺含程式/技術筆試）。
3. **一次或多次技術 + 用人主管面試**（技術深掘 + 專案經驗）。
4. **HR 面試**（人格特質、動機、薪資談判）。
5. **體檢 / 發 offer / 報到**。
