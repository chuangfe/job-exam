## 📚 核心能力
- **程式基礎**：資料結構（陣列/雜湊/堆疊/佇列/樹/圖/堆積）、演算法（排序、二分、雙指標、BFS/DFS、DP、貪婪）、時間/空間複雜度分析。
- **系統知識**：
  - **作業系統**：行程/執行緒、排程、記憶體管理、死結、同步互斥。
  - **計算機網路**：OSI/TCP-IP、HTTP/HTTPS、DNS、TCP vs UDP。
  - **資料庫**：正規化、索引、交易 ACID、鎖、查詢優化（製造業資料量大，效能很重要）。
- **工程實務**：Git 工作流、單元/整合測試、CI/CD pipeline、Docker 容器化、Linux 環境操作、Code Review 文化。
- **領域專長（依職位）**：
  - **後端/軟體**：Web 服務、API 設計（REST）、框架（Spring/.NET）。
  - **資料工程**：SQL 進階、ETL、資料管線、大數據工具（Spark/Kafka）。
  - **資安**：OWASP Top 10、身分驗證、加密、網路防護。
  - **智慧製造/自動化**：與產線資料整合、自動化腳本、基礎 ML/資料分析。

## 🗓️ 學習順序（建議 8–12 週）
1. **第 1–2 週｜語言 + 資料結構基礎**：選定主力語言補到熟，同步開始刷 LeetCode Easy（見練習題目）。
2. **第 3–4 週｜SQL + 資料庫**：練 LeetCode Database 題、補正規化/索引/交易觀念。
3. **第 5–6 週｜演算法主題刷題**：依主題系統性刷 Medium（雙指標、樹、圖、DP）。
4. **第 7–8 週｜工程實務 + Linux**：Git/測試/Docker/CI-CD 實作，用一個小專案串起來。
5. **第 9–10 週｜實作專案**：完成 1–2 個可放履歷的專案（見練習題目），含 Git/測試/容器化。
6. **第 11–12 週｜面試衝刺**：性向測驗練習、專案深掘演練、行為面試 STAR、模擬面試。

> 提醒：優先補齊 JD「必備條件」缺口，再追加分項目；專案 > 純刷題，更能展現工程實力。
