## 🛠️ 實作專案建議（比刷題更能展現實力，可放履歷/作品集）
1. **製造/監控數據儀表板（資料工程 + 後端方向）**
   - 技術棧：Python（FastAPI/Flask）+ PostgreSQL/MS SQL + 前端圖表（React 或 Streamlit）。
   - 重點：模擬產線/感測器資料 → ETL 寫入資料庫 → API 提供查詢 → 即時圖表；加上 Docker 容器化與 GitHub Actions CI。
   - 對齊 JD：資料管線、SQL、後端 API、容器化、工程實務。

2. **企業內部工具 / 工單系統（後端 + 全端方向）**
   - 技術棧：Java(Spring Boot) 或 C#(.NET) + 關聯式資料庫 + REST API + 簡單前端。
   - 重點：使用者驗證/權限、CRUD、單元測試、Swagger 文件、Docker 部署。
   - 對齊 JD：企業系統開發、API 設計、測試、資料庫設計。

3. **自動化 / 智慧製造小工具（自動化 + 資料分析方向）**
   - 技術棧：Python（pandas、scikit-learn）+ 排程（cron/Airflow）。
   - 重點：自動抓取/清理資料、產生報表或簡單良率預測模型、用 Linux 排程定時執行、log 與錯誤處理。
   - 對齊 JD：自動化、資料分析、Linux、基礎 ML。

> 專案撰寫提醒：README 寫清楚動機/架構/如何執行；commit 歷史乾淨；附測試與 Docker；履歷以**量化成果**描述（見履歷準備）。
