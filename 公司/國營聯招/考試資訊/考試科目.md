# 國營聯招（資訊類師級）- 考試科目詳細說明

> ⚠️ 各年度科目組合可能微調，本頁列出**常見組合**，務必以**當年度官方簡章**為準。

## 📊 配分結構

國營聯招採「第一試（筆試）+ 第二試（口試/現場測試）」兩階段：

### 第一試：筆試（佔總成績較高，常見約 80%）
- **共同科目**：國文（含作文與閱讀測驗）、英文
- **專業科目**：資訊類師級通常為 **3 科專業科目**
- **配分**：共同科目佔比較低、專業科目佔比較高（常見共同 20% + 專業 80%，**確切比例依簡章**）

### 第二試：口試（佔總成績較低，常見約 20%）
- 多為個別口試（部分年度/機構可能加入專業實作或測驗）
- **確切佔比與形式依當年度簡章與分發機構而定**

### 錄取門檻
- 各科或總成績常設有**最低標準**（如專業科目缺考或零分不予錄取等），依簡章規定。
- 最終依**總成績由高至低、配合志願與缺額分發**。

## 📖 共同科目

### 國文（含作文與閱讀測驗）
- **題型**：作文（占國文較大比重）＋ 閱讀測驗（選擇題）
- **準備方向**：
  - 作文：練習論說文結構（破題—論點—舉例—結論），題目多為職場態度、社會議題、價值觀類。
  - 閱讀測驗：練習文意理解、語詞辨析、文言與白話閱讀。
- **CP 值**：作文短期難大幅提升，但有基本盤；不必花過多時間，求穩不失分。

### 英文
- **題型**：選擇題（字彙、文法、克漏字、閱讀測驗）
- **難度**：約大學英文／多益中等程度
- **準備方向**：
  - 累積資訊領域常用單字（algorithm, database, security…）。
  - 練習多益或全民英檢中高級題型。
  - 閱讀短篇英文資訊／科技文章維持手感。

## 💻 專業科目（資訊類師級常見組合）

> 常見三科：**① 資料庫應用、② 資訊管理與資通安全、③ 計算機概論（或程式設計／系統分析）**。實際科目名稱與數量**依當年度簡章**，近年資訊管理類常見「資料庫應用」與「資訊管理（含資通安全）」為核心。

### 專業科目一：資料庫應用
**重要度：★★★★★**
- **資料庫概念**：DBMS 架構、三層綱要、資料獨立性
- **資料模型**：ER Model、關聯式模型、關聯代數
- **正規化**：1NF / 2NF / 3NF / BCNF，與反正規化的取捨
- **SQL**：DDL / DML / DCL、JOIN（INNER/LEFT/RIGHT/FULL）、子查詢、GROUP BY / HAVING、聚合函數、檢視表（View）
- **交易管理**：ACID、並行控制（鎖定、二階段鎖定 2PL）、隔離級別
- **索引與效能**：B+ Tree 索引、雜湊索引、查詢最佳化基本概念
- **新興議題**：NoSQL、資料倉儲、OLAP/OLTP 差異、大數據基本概念

### 專業科目二：資訊管理與資通安全
**重要度：★★★★★**
- **資訊管理（MIS）**：
  - 系統分析與設計（SDLC、瀑布式 vs 敏捷、UML 基本概念）
  - 資訊系統類型（TPS、MIS、DSS、ERP、SCM、CRM）
  - 專案管理（範疇/時程/成本、甘特圖、要徑法 CPM）
  - 決策支援、企業流程再造、IT 治理（COBIT、ITIL 概念）
- **資通安全**：
  - CIA 三要素（機密性、完整性、可用性）
  - 密碼學：對稱（AES/DES）、非對稱（RSA）、雜湊（SHA）、數位簽章、PKI
  - 常見攻擊：SQL Injection、XSS、CSRF、DDoS、社交工程、釣魚、勒索軟體
  - 防護機制：防火牆、IDS/IPS、VPN、SSL/TLS、存取控制
  - 管理面：ISO 27001 資安管理、風險管理、個資法概念

### 專業科目三：計算機概論 / 程式設計 / 系統分析（依年度擇一）
**重要度：★★★★☆**
- **計算機概論**：計算機組織、作業系統（行程/記憶體/排程/死結）、資料結構、網路（OSI/TCP-IP）、新興科技（雲端、IoT、AI）
- **程式設計**：基本語法、控制結構、函數與遞迴、OOP、資料結構實作、常見演算法
- **系統分析與設計**：需求分析、資料流程圖（DFD）、ER 圖、UML、軟體開發方法論

## 📊 成績計算（概念）
```
總成績 = 第一試（筆試）× 約80% + 第二試（口試）× 約20%
其中筆試 = 共同科目 × 約20% + 專業科目 × 約80%
（以上比例為常見值，務必以當年度簡章為準）
```
