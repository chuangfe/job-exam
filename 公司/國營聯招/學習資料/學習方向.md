# 國營聯招（資訊類師級）- 學習方向

> 列出「**該學什麼、順序、深度重點**」。對應 [考試科目](../考試資訊/考試科目.md)。
> 資訊類師級核心是 **資料庫應用** + **資訊管理與資通安全** + **計算機概論／程式設計**，與中華電信偏「計概＋程式」的組合不同，**資管與資安比重明顯更高**。

## 🎯 優先序總覽
```
第一優先（必讀，分數主力）：
- 資料庫應用：SQL + 正規化 + 交易管理        ★★★★★
- 資訊管理：系統分析設計 + 資訊系統類型 + 專案管理 ★★★★★
- 資通安全：密碼學 + 常見攻擊與防護           ★★★★★

第二優先（重要，廣度科目）：
- 計算機概論：作業系統 + 資料結構 + 網路       ★★★★☆
- 程式設計／系統分析（依年度而定）            ★★★★☆

第三優先（穩盤，勿過度投入）：
- 英文（資訊單字 + 多益題型）                ★★★☆☆
- 國文（作文結構 + 閱讀測驗）                ★★★☆☆
```

## 📚 各科學習重點與深度

### 1. 資料庫應用（分數主力，務必精熟）
- **SQL（最高 CP 值，務必練到能默寫）**
  - SELECT 全套：WHERE、GROUP BY、HAVING、ORDER BY、DISTINCT
  - JOIN：INNER / LEFT / RIGHT / FULL，能畫文氏圖說明差異
  - 子查詢（相關 vs 非相關）、EXISTS、IN
  - 聚合函數（COUNT/SUM/AVG/MAX/MIN）與 NULL 的處理
  - DDL（CREATE/ALTER/DROP）、DML、DCL（GRANT/REVOKE）
- **正規化（必考計算題）**：能對給定資料表做 1NF→2NF→3NF→BCNF，並說明各 NF 消除的異常（插入/刪除/更新異常）。
- **ER Model**：實體、屬性、關係、基數（1:1, 1:N, M:N）、弱實體；ER 圖轉關聯表。
- **交易管理**：ACID 逐字會解釋；並行問題（lost update、dirty read、unrepeatable read、phantom）；鎖定與 2PL；隔離級別。
- **索引與效能**：B+ Tree 索引運作、何時該建索引、索引的代價。
- **深度提醒**：資料庫是資訊類師級**最穩定的得分區**，務必先把這科讀到 80 分以上再分配時間給其他科。

### 2. 資訊管理（MIS，背誦與架構為主）
- **系統開發生命週期（SDLC）**：各階段任務；瀑布式 vs 雛型法 vs 敏捷（Agile/Scrum）的差異與適用情境。
- **系統分析工具**：DFD（資料流程圖）符號與層次、ER 圖、UML（類別圖、使用案例圖、循序圖概念）。
- **資訊系統類型**：TPS、MIS、DSS、EIS、ERP、SCM、CRM、KM——能說出各自定位與服務的管理層級。
- **專案管理**：範疇/時程/成本三角、WBS、甘特圖、要徑法（CPM，會算最早/最晚時間與要徑）、PERT 概念。
- **IT 治理與管理**：ITIL、COBIT、BPR（企業流程再造）、IT 與企業策略對齊的概念。
- **深度提醒**：MIS 偏記憶與架構理解，靠製作比較表與心智圖最有效率。

### 3. 資通安全（與資管常合併命題，比重高）
- **基礎**：CIA 三要素、鑑別（Authentication）/ 授權（Authorization）/ 不可否認性。
- **密碼學（必考）**：
  - 對稱式（AES、DES）vs 非對稱式（RSA）——能比較金鑰數量、速度、用途。
  - 雜湊（SHA 系列、MD5 的弱點）、數位簽章流程、PKI 與憑證。
- **常見攻擊與防護**：SQL Injection、XSS、CSRF、DDoS、社交工程、釣魚、勒索軟體、中間人攻擊——每個都要能講「原理 + 防護方法」。
- **網路與系統防護**：防火牆、IDS/IPS、VPN、SSL/TLS、存取控制模型（DAC/MAC/RBAC）。
- **管理面**：ISO 27001（ISMS）、風險評估流程、個資法基本概念。

### 4. 計算機概論（廣度科目，求穩不失分）
- **作業系統**：行程 vs 執行緒、排程演算法（FCFS/SJF/RR/Priority，能手算）、記憶體管理（分頁/分段/虛擬記憶體、頁面置換 FIFO/LRU/Optimal）、死結四條件與銀行家演算法。
- **資料結構**：陣列/鏈結串列/堆疊/佇列；樹（BST、AVL、Heap、B/B+ Tree）；圖（DFS/BFS、最短路徑、最小生成樹）；**各操作時間複雜度務必背熟**。
- **網路**：OSI 七層 vs TCP/IP、TCP vs UDP、三向交握、IP/子網路遮罩計算、常見協定（HTTP/DNS/DHCP/SMTP）。
- **計算機組織**：CPU、記憶體階層、Cache 命中率、管線化。
- **新興科技**：雲端（IaaS/PaaS/SaaS）、虛擬化與容器、IoT、AI/大數據基本概念。

### 5. 程式設計（若該年度列入）
- 基本語法、控制結構、函數與遞迴、OOP 四大特性、常見演算法（排序/搜尋）與資料結構實作。
- 建議擇一熟悉語言（C/C++/Java/Python），能讀懂程式碼並預測輸出、找錯。

### 6. 共同科目
- **英文**：每天累積資訊單字 + 多益／全民英檢中高級題型，維持手感即可，勿過度投入。
- **國文**：作文先把論說文架構練熟（破題、論點、舉例、收尾），閱讀測驗靠平時累積；求基本盤、不失分。

## 🗓️ 讀書順序建議（約 6 個月）

1. **階段一（第 1-2 月）打底**：資料庫應用（SQL+正規化）＋ 計算機概論基礎（OS、資料結構）。
2. **階段二（第 3-4 月）拓展**：資訊管理（系統分析、專案管理）＋ 資通安全（密碼學、攻防）＋ 網路。
3. **階段三（第 5 月）整合**：把資管與資安交叉複習（兩者常合併命題），補程式設計／系統分析。
4. **階段四（第 6 月，考前衝刺）**：大量做國營聯招歷屆 + 高普考資訊處理考古題，限時模擬、整理錯題本，國文作文與英文做最後手感維持。

## 💡 與中華電信準備的差異提醒
- 中華電信偏「計概 + 程式設計手寫」；國營聯招資訊類**資管與資安是大宗**，且**資料庫（SQL/正規化）幾乎必拿分**。
- 若同時準備兩者，**資料庫、作業系統、資料結構、網路、資安**是共用基礎，可一次讀透；資管（系統分析、專案管理）則是國營聯招要額外加強的部分。
