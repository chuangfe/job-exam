# Google - Software Engineer 總覽

> 範本類型：**外商科技**（LeetCode + 系統設計 + 行為面試）。

## 🏢 公司簡介
- **公司名稱**：Google（Alphabet 旗下）
- **產業 / 規模 / 在台據點**：全球大型科技公司（搜尋、廣告、雲端 GCP、Android、YouTube 等）。台灣設有研發據點（台北 101、新北板橋、新竹等），是亞太重要的硬體與軟體研發中心。
- **文化關鍵字**：**Googleyness & Leadership**——協作（collaboration）、面對模糊（comfort with ambiguity）、使用者導向（user-focus）、謙遜與好奇心、bias to action。

## 💼 目標職位
- **職位 / 職級**：Software Engineer（常見 L3 = SWE II 新鮮人/初階、L4 = SWE III 中階、L5 = Senior SWE）
- **工作內容**：設計、開發、測試、部署與維護大規模軟體系統；跨團隊協作、code review、解決有模糊性的技術問題。

## 🔄 面試流程（關卡）
1. **履歷 / 內推（Referral）** — 內推命中率明顯高於海投，優先取得。
2. **Recruiter screen** — 招募人員電話初談（背景、期望、流程說明）。
3. **線上 / 電話技術面（1-2 輪 coding）** — 共享編輯器（Google Docs / 線上 IDE）做演算法題。
4. **Onsite / Virtual onsite（通常 4-5 輪）** — coding × 3-4 輪 + 一輪偏 **behavioral / Googleyness**；**資深職位（L5+）加 System Design**。
5. **Hiring Committee 審核** — 由委員會（非面試官）綜合評分卡決定是否通過，避免單一面試官偏誤。
6. **Team match** — 通過後與有缺的團隊媒合（可能需與多個 hiring manager 面談）。
7. **Offer** — 薪酬團隊核定並發出 offer。
- **時程**：整體常耗時**數週到數月**，team match 階段可能拉長等待時間。

## 💰 薪資結構
- **組成**：Base（基本薪資）+ **RSU**（限制性股票，通常 4 年 vesting）+ **Bonus**（年度績效獎金，與評等掛鉤），部分職位有 sign-on bonus。
- **參考來源**：實際數字依職級、地區、議價而異，請參考 **[levels.fyi](https://www.levels.fyi/companies/google/salaries/software-engineer)**（依 L3/L4/L5 與地點查詢），勿憑空填寫精確數字。
